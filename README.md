# gameboff
A GameBoy emulator written in C.

## Running 
### 1. Clone repo
```
git clone https://github.com/tmkcell/gameboff.git && cd gameboff
```
### 2. Compile with meson build system (don't forget to install SDL3/SDL2)
```
meson setup build
meson compile -C build
meson install -C build
```
